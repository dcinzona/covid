## Displaying COVID-19 Viral Spread over time

### Requirements:
* NodeJS v12+

### Instructions
1. Install NodeJS LTS or Current: https://nodejs.org/en/
2. Clone this repo
3. `cd ./covid`
4. `npm install`
5. `node index.js`
6. Go to http://localhost:3000
7. Click the `play` button on the bottom left
